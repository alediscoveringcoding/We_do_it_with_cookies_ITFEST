## Color Palette Reference

```
Primary Colors:
  --teal: #0d9488
    Used for: Main buttons, links, badges, progress bar

  --teal-light: #14b8a6
    Used for: Button hover states, active states

  --teal-pale: #ccfbf1
    Used for: Badge backgrounds, light accents

Secondary:
  --green: #2d6a4f
    Used for: Dashboard gradient, secondary brand

Neutral:
  --dark: #1a1a2e (headings, main text)
  --mid: #374151 (secondary text)
  --soft: #6b7280 (tertiary text)
  --bg: #f9fafb (page background)
  --border: #e5e7eb (dividers, card borders)
```

## Animation Examples

### Card Swipe Out (No)

```
Initial: [Card in center] opacity: 1
Step 1:  [Card moves left] opacity: 0.8
Step 2:  [Card exits left] opacity: 0
Time:    300ms, smooth easing
```

```
Initial: [Card offscreen below] scale: 0.95, opacity: 0

Time:    300ms, smooth easing
```

**Visual Design Notes:**

✅ Consistent spacing (multiples of 4px)
✅ Rounded corners (4px, 8px, 16px)
✅ Shadows match design system
✅ Animations are smooth (300ms)
✅ Touch targets ≥ 44x44px on mobile
✅ Text has good contrast (AA standard)
✅ Fonts: Playfair (headings), DM Sans (body)
✅ All interactive elements have hover states
✅ Keyboard focus indicators visible
✅ Colors support colorblind users (avoiding red-green only)
